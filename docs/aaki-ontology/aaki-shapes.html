AAKI Shapes

Namespace: http://open-services.net/ns/aaki/shapes#  |  Inherits from: http://open-services.net/ns/am/shapes/3.0#ResourceShape via oslc:superShape

OSLC ResourceShapes constraining the AAKI vocabulary for a server-side API contract. Every AAKI shape declares oslc:superShape <http://open-services.net/ns/am/shapes/3.0#ResourceShape> to inherit the common OSLC core properties (title, description, creator, created, modified, serviceProvider, instanceShape, plus the six common link types derives / elaborates / refine / external / satisfy / trace) from the OASIS-standard AM Resource shape. AAKI shapes themselves carry only the AAKI-specific properties.

See also: class diagram, stage-lifecycle sequence diagram.
